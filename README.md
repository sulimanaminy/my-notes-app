# My Notes App

A simple web application for creating, managing, and organizing personal notes. Built with React to practice state management, component-based architecture, and modern frontend development techniques.

## Features

- Create, edit, and delete notes.
- Responsive design for mobile and desktop.
- State management using React hooks.
- Basic form validation for note input.

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
